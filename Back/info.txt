Para instalar todas as dependências é necessário digitar: npm install no powerShell

dependências utilizadas:
  - npm install express --save
  - npm install mysql
  - npm install nodemon -g <------- único necessário instalar, o resto só dar npm install para poder instalar tudo
  - npm install --save sequelize
  - npm install --save mysql2


O nodemon serve para atualizar o server a qualquer mudança, não é necessário ficar abrindo e fechando o servidor.
Para acessar pelo nodemon é só escrever: nodemon start - informação pode ser modificada no package.json
Como o nodemon executa um script detro do powerShell, é necessário abrir o powerShell como administrador e executar:
Set-ExecutionPolicy Remotesigned
Assim é permitido utilizar scripts assinados digitalmente
